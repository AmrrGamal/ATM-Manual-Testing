# ATM System Testing & Database Testing Portfolio

## Overview
This repository demonstrates a complete Manual Software Testing project for a Banking ATM System following the Software Testing Life Cycle (STLC). It covers requirements analysis, test planning, test design, execution, defect reporting, database validation, and final testing reports.

## Project Highlights

- Complete STLC implementation
- Functional Testing
- Negative Testing
- Input Validation Testing
- Error Handling Testing
- Security Testing
- Performance Testing
- Reliability Testing
- Usability Testing
- Database Testing using SQL
- Requirements Traceability Matrix (RTM)
- Professional QA Documentation
